#  WowFood â€“ Online Food Ordering System

WowFood is a web-based **Online Food Ordering System** developed as part of the **Rapid Application Development (RAD)** module at **Wayamba University of Sri Lanka**.  
The system enables customers to browse food items, place orders, and communicate with restaurants, while administrators manage products, orders, users, and customer messages through a secure admin panel.

---

##  System Features

###  Customer Features
- User registration and login
- Browse available food items
- Add food items to cart
- Update cart and place orders
- Contact restaurants via message system
- Secure checkout process

###  Admin Features
- Secure admin authentication
- Admin dashboard overview
- Manage food products (add, update, delete)
- Manage customer orders
- View and respond to customer messages
- Manage registered users

---

##  Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server Environment:** Apache (XAMPP / WAMP)
